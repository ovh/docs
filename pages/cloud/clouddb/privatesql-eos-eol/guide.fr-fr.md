Les produits couverts par ces annonces de fin de vente et de fin de vie sont les services Cloud Databases Web Hosting (également connus sous le nom de services SQLPrivé), instances SGBD joignables via le réseau Web Hosting. Référez-vous à la [politique de fin de vie des bases de données managées](../managed-db-life-cycle-policy/) pour plus d'informations.
